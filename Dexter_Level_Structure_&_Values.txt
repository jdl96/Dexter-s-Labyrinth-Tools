Level Tile Values:

1 = Red Wall
2 = Empty
5 = Light Brown Sand
6 = Blue Sand
7 = Green Sand
8 = Dark Brown Sand
9 = Blue Wall

Object Values:

Object Usage Example: birthPlayer(me, 12, 7)

birthPlayer = Dexter
birthDoor = Level Exit
birthDiamond = Microchip
birthRandy = Random-turning Virus
birthLefty = Left-turning Virus
birthRighty = Right-turning Virus
